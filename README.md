# QualONumero - Spec-Driven Development (SDD)

Este repositório contém a especificação completa para o desenvolvimento do jogo casual **QualONumero**, utilizando a metodologia de *Spec-Driven Development* (Desenvolvimento Guiado por Especificações). 

O objetivo do projeto é orientar um agente de IA a construir uma aplicação frontend completa, funcional e padronizada sem a necessidade de intervenção manual no código.

## 📁 Estrutura do Repositório

O projeto é guiado pelos seguintes documentos informativos:

* **`AGENTS.md`**: Define as regras rígidas de comportamento da IA, padrões de código limpo, tratamento de erros e fluxo de trabalho linear.
* **`specs.md`**: Contém a visão geral do jogo, fluxo das telas, regras de negócio (sistema de 5 vidas) e as prioridades de desenvolvimento.
* **`design.md`**: Estabelece a identidade visual do app, paleta de cores calmas, tipografia e as microinterações (como a animação de erro no input).

## 🎮 Sobre o Jogo
O **QualONumero** é um jogo rápido de adivinhação web onde o usuário tem até 5 tentativas para acertar um número aleatório gerado entre 1 e 100, ideal para passar o tempo de forma simples e intuitiva.
